# Student Performance Tracker

A portfolio-ready web application for tracking course records, GPA estimates,
academic trends, and QA automation coverage.

This project is designed for applications to:

- QA Automation Engineer
- Junior Software Developer
- Technical Business Analyst
- IT Business Analyst

## Features

- Course record table based on software engineering transcript data.
- Weighted average and GPA estimate.
- Academic trend chart by semester.
- Skill profile by course category.
- Add and delete course records from the UI.
- Automated Playwright tests for key user flows.

## Tech Stack

- React
- TypeScript
- Vite
- Recharts
- Playwright
- GitHub-ready project structure

## QA Scope

Automated tests cover:

- Dashboard rendering and summary metrics.
- Adding a new course record.
- Score-to-grade mapping.
- Deleting a course record.
